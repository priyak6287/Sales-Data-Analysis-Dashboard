Sales Data Analysis Dashboard

This project presents an end-to-end sales data analysis using Microsoft Excel. The dataset was cleaned and analyzed using pivot tables, and an interactive dashboard was created to visualize key business insights.

Project Overview

The dashboard helps analyze sales performance across different categories, regions, and time periods. It highlights top-performing products and monthly sales trends to support data-driven decision making.

Key Features

- Sales performance by category
- Region-wise sales analysis
- Monthly sales trend visualization
- Top 10 products by sales
- Sub-category performance analysis

Tools Used

- Microsoft Excel (Pivot Tables & Charts)
